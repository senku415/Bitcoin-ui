# Glassmorphism Crypto & Investment Cards

A visually stunning, modern dashboard card interface featuring a dark mode aesthetic with vibrant glassmorphism glow effects. This project showcases responsive design using CSS Grid and Flexbox, complete with smooth gradients and backdrop filters.

## 🚀 Features

*   **Glassmorphism UI**: Uses `backdrop-filter` and translucent borders to achieve a frosted-glass effect.
*   **Vibrant Glow Effects**: Custom radial gradients and matching box-shadow glows for Purple (Savings), Green (Bitcoin), and Blue (Stocks) themes.
*   **Fully Responsive**: Built with CSS Grid, automatically adapting from a multi-column desktop layout to a sleek, single-column mobile view (under `520px`).
*   **Font Awesome Integration**: Clean, modern iconography for financial metrics.

## 🛠️ Tech Stack

*   **HTML5**: Semantic structure.
*   **CSS3**: Custom grid configurations, transitions, media queries, and glassmorphism styling.
*   **Font Awesome**: Scalable vector icons.
